# Number Recognizer (kNN) – Frontend Only

## Описание
Това е интерактивен **Number Recognizer**, реализиран изцяло в браузъра с **JavaScript**.  
Проектът използва **k-Nearest Neighbors (kNN)** алгоритъм за разпознаване на ръкописни цифри от 0 до 9.  

Докато потребителят рисува цифра на Canvas, системата в **реално време** изчислява процентната прилика с всяка цифра и визуализира резултатите с прогрес барове.  
Това демонстрира както алгоритмично мислене, така и **advanced JavaScript** техники (Canvas API, асинхронни изчисления, динамичен UI).

## Функционалности
- Canvas за интерактивно рисуване
- Preprocessing на изображението: resize, grayscale, нормализация
- Имплементация на kNN алгоритъм „от нулата“
- Динамична визуализация на проценти прилика за 0–9
- Показване на топ 3 най-близки примера от обучаващ dataset

## Как да използвате
1. Отворете `index.html` в браузъра (не се изисква сървър).  
2. Рисувайте цифра в Canvas.  
3. Вижте процента прилика и класа на цифрата в реално време.  

## Технологии
- HTML5 Canvas API
- Vanilla JavaScript (ES6+)
- CSS за визуализация
- JSON dataset за обучаващи примери

## Цел на проекта
- Демонстриране на алгоритмично мислене с JavaScript
- Реализация на machine learning алгоритъм (kNN) без външни библиотеки
- Динамична обработка и визуализация на данни в браузъра
